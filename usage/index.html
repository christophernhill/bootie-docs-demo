Usage

April 28, 2015

Add more document pages

You can create more pages by hugo new command.

If you want to quick access for the page, you can include it in “main menu” of your site by configuring it as params.mainMenu in your config.toml (or config.yaml):

# config.toml
[params]
  mainMenu = ["<your-page-name>", ...]

Then you can access the page by link in navigation bar on top of the site.

It is recommended to create the page on top of your content/ directory if you want to access it by navigation bar.
